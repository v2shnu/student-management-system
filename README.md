# Student Management System (Python)

A command-line Student Management System built using Python.

## Features
- Add, view, search, update, and delete student records
- Persistent storage using CSV file
- Modular code structure

## Technologies Used
- Python
- File handling (CSV)

## Project Structure
- main.py — menu and app flow
- student_ops.py — student operations
- data.py — file handling logic

## How to Run
1. Clone the repository
2. Run `main.py`
3. Use the menu to manage students
